Kpizza

An application which demonstrate the use of NodeJs, Express and MongoDb to build a shopping cart.

This application consists of two main components

The server component use NodeJs Express and MongoDb to offer three main application programming interfaces.

  • The customer API : This API to register users to the application and an admin only API to manage customers.
  • The Order API : This API manage the full life-cycle of a placed order.
  • The Pizza API : This API is mainly the content provided for the web site and allow admin to add, update and delete Pizza types.
